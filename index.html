Aprendendo HTML (<h1>)

1. Tags (<h2>)

No contexto HTML, as tags são elementos usados para estruturar e organizar o conteúdo de uma pégina web. Cada tag é uma instrução que informa ao navegador como exibir determinado conteúdo ou como ele deve se comportar. As tags são a base da linguagem HTML e são representadas por nomes envolvidos por colchetes angulares (< >).

1.1. Estrutura de uma tag

A maioria das tags têm uma abertura e um fechamento, formando um elemento HTML

  • <p>: tag de abertura de um paragrafo
  • </p>: tag de fechamento de um paragrafo

Exemplo de uma tag com abertura e fechamento:

<p> Este é um paragrafo</p>

OBS: Algumas tags não possuem fechamento (autocontidas). Dessa forma elas não tem conteudos dentro delas.

1.2.1. Tags estruturais

As tags estruturais no HTML são usadas para definir a estrutura básica e a organização lógica de uma página web. Elas ajudam a segmentar o conteúdo em seções claras e semânticas, facilitando a compreensão por navegadores, motores de busca e leitores de tela.

Essas tags são fundamentais para o design e a acessibilidade de uma página, pois permitem organizar o conteúdo hierarquicamente e melhorar a navegação.

  • <html>: Define o documento HTML como o todo.

  • <head>: Contém metadados e informações gerais sobre a página (não visíveis ao usuário).

  • <body>: Contém o conteúdo visível da página.

  • <header>: Representa a seção de cabeçalho de uma página ou de uma seção.

  • <nav>: Indica uma área de navegação, como menus e links.

  • <main>: Representa o conteúdo principal do documento.

  • <section>: Define seções de conteúdo relacionadas.

  • <article>: Indica um conteúdo independente e completo, como postagens de blog ou artigos.

  • <aside>: Representa informações relacionadas ao conteúdo principal, como barras laterais ou caixas de informações extras.

  • <footer>: Representa o rodapé da página ou de uma seção.

1.2.2. Tags de texto

As tags de texto no HTML são usadas para formatar, estruturar e exibir conteúdos textuais em uma página web. Elas ajudam a organizar o texto em parágrafos, títulos, listas e permitem aplicar estilos específicos, como negrito, itálico ou sublinhado.

Essas tags também possuem uma função semântica, indicando o propósito ou a importância de um determinado texto para o navegador, motores de busca e leitores de tela.

  • <h1> ... <h6>: Representam títulos e subtítulos de diferentes níveis de importância.

  • <p>: Define um parágrafo de texto.

  • <strong>: Dá ênfase semântica ao texto, geralmente exibindo-o em negrito.

  • <b>: Apenas aplica o estilo negrito, sem ênfase semântica.

  • <em>: Dá ênfase semântica ao texto, geralmente exibindo-o em itálico.

  • <i>: Apenas aplica o estilo itálico, sem ênfase semântica.

  • <hr>: Insere uma linha horizontal para dividir conteúdos.

  • <pre>: Exibe o texto exatamente como ele foi escrito, incluindo espaços e quebras de linha.

  • <br>: Insere uma quebra de linha no texto.

  • <blockquote>: Usada para citar um texto em bloco.

    • Lorem, ipsum dolor sit amet consectetur adipisicing elit. Vitae sequi a maxime saepe recusandae? Eligendi harum, nam aliquid odit, maxime pariatur, dolorem quo aut eius laborum esse amet atque sequi.

  • <q>: Usada para citações curtas, que aparecem entre aspas.

  • <ul>: Lista não ordenada (com marcadores)..

  • <ol>: Lista ordenada (com números ou letras).

  • <li>: Define os itens da lista.

  • <small>: Exibe texto em tamanho reduzido.

  • <u>: Sublinha o texto.

  • <s>: Exibe texto riscado sem indicar remoção semântica.

  • <del>: Indica texto excluído. Essa parte foi excluida.

  • <sup>: Texto elevado (superior) --> x2

  • <sub>: Texto rebaixado (infeir) --> Xi

  • <code>: Exibe trechos de código no texto.

    • Use a tag <html> no início do documento.

  • <ins>: Indica texto inserido, geralmente exibindo-o sublinhado.

1.2.3. Tags de link e navegação

As tags de link e navegação no HTML são usadas para criar links e organizar a navegação de uma página ou site. Elas permitem que os usuários se desloquem para outras páginas, seções específicas, ou até mesmo interajam com recursos externos, como documentos ou arquivos.

  • <a>: É usada para criar links clicáveis. tendo como atributo mais importante é o href, que especifica o destino do link.
    • Link para página do google Link para o inicio desta página Link para baixar um arquivo
